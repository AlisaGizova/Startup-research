# Исследование стартапов

**Задачи проекта:** На основе исторических данных об инвестициях и стартапах подготовить аналитический датасет, провести его исследование и предобработку для формирования первоначального понимания рынка и выявления ключевых закономерностей.

**Описание проекта:** Проведена загрузка и предобработка данных из семи таблиц, содержащих информацию о компаниях, раундах финансирования, сделках по покупке, сотрудниках и фондах. Выполнен исследовательский анализ, в ходе которого определены категории стартапов с наиболее высокими и вариативными ценами покупок, рассчитаны типичные суммы финансирования и количество раундов в зависимости от статуса компании, выявлены тренды по годам. Сформулированы выводы и практические рекомендации для дальнейшего анализа инвестиционного рынка.
